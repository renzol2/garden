# Turborepo

Turborepo is a high-performance build system for [[JavaScript]] and [[TypeScript]] monorepos. It supports the following features:

- incremental builds
- content-aware hashing
- cloud caching
- parallel execution
- task pipelines
- zero runtime overhead
- pruned subsets
- JSON configuration
- profile in your browser
