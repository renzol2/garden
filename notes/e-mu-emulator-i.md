# E-Mu Emulator I

The **E-Mu Emulator I** is an early, [[digital-audio|digital]] [[sampling-composition|sampling]] instrument invented in 1981.

- $10,000, 66lbs
- Had a hard drive to store sample data
- 128 kB of sample memory
